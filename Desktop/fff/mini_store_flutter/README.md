# تطبيق متجر صغير - Flutter

تطبيق متجر إلكتروني متكامل مبني بـ Flutter و Dart مع جميع المميزات الحديثة.

## المميزات

✅ **المصادقة والحسابات**
- تسجيل دخول وإنشاء حسابات جديدة
- إدارة بيانات المستخدم
- تسجيل الخروج الآمن

✅ **المنتجات والبحث**
- عرض جميع المنتجات
- تصنيف المنتجات حسب الفئات
- البحث عن المنتجات
- عرض تفاصيل المنتج

✅ **السلة والشراء**
- إضافة المنتجات للسلة
- تحديث الكمية
- حذف المنتجات
- حساب السعر الإجمالي
- إتمام الشراء

✅ **المفضلة**
- إضافة المنتجات للمفضلة
- حذف من المفضلة
- عرض قائمة المفضلة

✅ **التخزين المحلي**
- حفظ السلة محلياً
- حفظ المفضلة محلياً
- حفظ بيانات المستخدم

✅ **الوضع الليلي والنهاري**
- تبديل بين الوضع الليلي والنهاري
- حفظ تفضيل المستخدم
- واجهة جميلة في كلا الوضعين

✅ **واجهة عربية**
- دعم كامل للغة العربية
- تصميم يناسب اللغات من اليمين لليسار

## المتطلبات

- Flutter SDK 3.0+
- Dart 3.0+
- Android Studio أو VS Code

## التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
cd mini_store_flutter
```

### 2. تثبيت المكتبات
```bash
flutter pub get
```

### 3. تشغيل التطبيق
```bash
flutter run
```

## المكتبات المستخدمة

- **provider**: إدارة الحالة
- **shared_preferences**: التخزين المحلي
- **http**: طلبات HTTP
- **firebase_core**: Firebase Core
- **firebase_auth**: Firebase Authentication
- **cloud_firestore**: Firestore Database
- **sqflite**: قاعدة بيانات محلية
- **cached_network_image**: تخزين الصور مؤقتاً
- **intl**: الدعم الدولي

## الشاشات الرئيسية

### 1. شاشة تسجيل الدخول
- إدخال البريد الإلكتروني وكلمة المرور
- رابط للذهاب لإنشاء حساب جديد
- معالجة الأخطاء والرسائل

### 2. شاشة إنشاء حساب
- إدخال الاسم والبريد والكلمة
- تأكيد كلمة المرور
- رابط للعودة لتسجيل الدخول

### 3. الشاشة الرئيسية
- عرض المنتجات في شبكة
- شريط بحث
- فلاتر الفئات
- شريط التنقل السفلي

### 4. شاشة السلة
- عرض المنتجات المضافة
- تحديث الكمية
- حذف المنتجات
- حساب الإجمالي
- زر إتمام الشراء

### 5. شاشة المفضلة
- عرض المنتجات المفضلة
- إزالة من المفضلة

### 6. شاشة الملف الشخصي
- عرض بيانات المستخدم
- تسجيل الخروج

## المميزات التقنية

### إدارة الحالة (State Management)
استخدام **Provider** لإدارة الحالة بشكل فعال:
- AuthProvider: إدارة حالة المصادقة
- CartProvider: إدارة حالة السلة
- FavoritesProvider: إدارة حالة المفضلة
- ThemeProvider: إدارة الثيم
- ProductsProvider: إدارة حالة المنتجات

### التخزين المحلي
استخدام **SharedPreferences** لتخزين:
- بيانات المستخدم
- السلة والمفضلة
- تفضيلات الثيم

### معالجة الأخطاء
- رسائل خطأ واضحة للمستخدم
- معالجة استثناءات الشبكة
- تحميل البيانات بشكل آمن
